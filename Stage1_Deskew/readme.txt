-----------------
Uses the following Python Libraries
-----------------
OpenCV: https://pypi.org/project/opencv-python/
Used for computer vision and image manipulation

Numpy: https://numpy.org/install/
Used for comprehensive mathematical functions


-----------------
To use this script
-----------------

Name the file original.jpg and put it into the same folder as the script

Select the corners of the image in the following order:
Top-Left
Top-Right
Bottom-Left
Bottom-Right

An image called deskewed.jpg will be saved
