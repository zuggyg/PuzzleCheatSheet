-----------------
Transcribing the DNA
-----------------
Every odd line should be describing the top edge (N,U,D)
Every even line should be describing the side edges (W,L,R)

N is for the top and bottom edge of the puzzle itself
W is for the left and right edge of the puzzle itself
U is for any knobs pointing up
D is for any knobs pointing down
L is for any knobs pointing left
R is for any knobs pointg right


-----------------
To use this script
-----------------

Make sure dnacut.txt is created correctly and in the folder of the script

The script will read each character to create an image.
If you desire different dimensions of each piece you will need to adjust the pstandard.png size

The script will output puzzlecut.png
